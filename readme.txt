TINQAD - Total Integrated Network for Quality Assurance and Development 2024

Features
*Display real-time data analytics and visualizations that permit users to easily formulate insights essential to decision-making;
*Consolidate and categorize the relevant data and resources of the team allowing for quick information access in a single location; and
*Automatically track the requirements needed in each task to monitor and disseminate information easily

Getting Started
To get started with TINQAD, simply follow these steps:
1. Open the website using the dilnet or VPN
2. Login using the Full Access Account: adminaccount@example.com; password
3. Create user accounts by registering a unique email 

To Copy TINQAD 
1. Clone the repository: git clone [repository URL]
2. Navigate to the project directory: cd [project directory]
3. Open the website by running: index.py
